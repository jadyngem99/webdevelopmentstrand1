Necessary Hardware and Software

In addition to paying for service through an Internet Service Provider (ISP), there is also different types of hardware and software that are needed to connect to the internet. Today computers can connect to the internet using cables, WiFi, and our cellphones.

Check out the table below that lists common hardware and software as well as their descriptions:

Common Hardware and Software Used to Access the Internet
Hardware/Software Description
Network Two or more computers connected for the purpose of sharing information.
Modem Allows one computer to connect to another computer and share/transfer data.
Router Device that routes data from a LAN to another network connection by local cable or a WiFI signal.
Ethernet Standard way to connect computers on a network over a wired connection. Also known by its technical name "IEEE 802.3, and the cable uses a RJ45 type of connector.
WiFi Wireless networking technology that utilizes radio waves to transmit information between computer devices. WiFi is simply a trademarked phrase that means IEEE 802.11x.
Browser Software that allows users to access and view web pages. Common browsers include Chrome, Firefox, Internet Explorer, and Safari.
World Wide Web (WWW) Collection of information that is accessible on the Internet using a system of interlinked hypertext documents.
Search Engine Website that is used to index/locate relevant websites when users enter search terms or phrases.

Anatomy and Explanation of a Uniform Resource Locator (URL)

A URL is an address of a specific webpage or file on the internet. It consists of three main parts:

  • Prefix - HTTP - the protocol
  • Domain - the server name
  • Path - the directory or directories that lead to and include the file
